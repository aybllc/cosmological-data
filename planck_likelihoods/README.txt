Data for Planck 2020 (NPIPE) likelihoods

Paper: Planck constraints on the tensor-to-scalar ratio
arxiv: https://arxiv.org/abs/2010.01139

Lollipop (lowlE, lowlB, lowlEB) covers the polarization data at large
angular scales (multipoles below 150).

Hillipop (TT, TE, EE, TTEETE) covers the temperature and polarization
data at samll angular scales (multipoles above 30).

The file cl_planck_lolEB_NPIPE_BB.dat contains bandpowers for the BB spectrum used for plotting only.
