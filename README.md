## Overview

This repository contains curated cosmological datasets for Hubble constant (H₀) measurements and systematic uncertainty analysis.

## Contents

### Data Sources

- **Pantheon+ Data**: Located in `Pantheon+_Data/`
  - Supernova cosmology measurements
  - Source: [PantheonPlusSH0ES/DataRelease](https://github.com/PantheonPlusSH0ES/DataRelease)

- **SH0ES Data**: Located in `SH0ES_Data/`
  - Distance ladder measurements
  - Cepheid calibrations

### Anchor Objects

- `NGC4258_HII_regions.csv` - HII region measurements for anchor galaxy NGC 4258
- `hosts_subset.csv` - Host galaxy properties
